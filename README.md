# 🎬 ReelsStudio - AI Reels Editörü

Malkoçoğlu Mobilya için özel olarak tasarlanmış, yapay zeka destekli video düzenleme ve müzik oluşturma platformu.

## ✨ Özellikler

- **🤖 AI Müzik Üretimi:** Videonuzun ruhuna uygun müziği sadece tarif ederek oluşturun.
- **📸 Akıllı Sahne Yönetimi:** Görsellerinizi sürükle-bırak yöntemiyle ekleyin ve sıralayın.
- **🎞 Canlı Önizleme:** Instagram Reels formatında anlık önizleme yapın.
- **📐 Gelişmiş Editör:** Her sahne için özel zoom, dikey hizalama ve metin bindirme ayarları.
- **🎶 Dinamik Müzik Kütüphanesi:** AI tarafından belirlenen kategoriye göre (Luxury, Energetic, Lofi vb.) otomatik müzik eşleşmesi.

## 🚀 Canlı Demo

Projeyi canlı olarak buradan inceleyebilirsiniz:
[https://erdemalpar.github.io/ReelsStudio/](https://erdemalpar.github.io/ReelsStudio/)

## 🛠 Kullanılan Teknolojiler

- HTML5 & CSS3 (Vanilla)
- JavaScript (ES6+)
- Tailwind CSS (Stil yönetimi için)
- Google Gemini API (AI Müzik Prompt Analizi için)
- Google Fonts (Playfair Display & Inter)

## 📦 Kurulum

Projeyi yerelinizde çalıştırmak için:

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/erdemalpar/ReelsStudio.git
   ```
2. `index.html` dosyasını tarayıcınızda açın.
